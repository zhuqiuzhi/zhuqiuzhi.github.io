title: 实现 Pic 函数
---

## 要求

它应当返回一个长度为 dy 的切片，其中每个元素是一个长度为 dx ，元素类型为 uint8 的切片。当你运行此程序时，它会将每个整数解释为灰度值（好吧，其实是蓝度值）并显示它所对应的图像

## 实现

```go
package main

import "golang.org/x/tour/pic"
import "math"

func Pic(dx, dy int) [][]uint8 {
	a := make([][]uint8, dy)
	for i := range a {
		b := make([]uint8,dx)
		for j := range b {
			b[j] = uint8(float64(i)*math.Log(float64(j)))
		}
		a[i]=append(a[i], b...)
	}
	return a
}

func main() {
	pic.Show(Pic)
}
```
